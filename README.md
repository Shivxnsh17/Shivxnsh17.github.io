# Equity Research Models

## Description
This repository contains a comprehensive collection of financial models and equity research tools built in Microsoft Excel. It covers various fundamental valuation methodologies, risk calculations, and cost of capital analyses for different companies and industries. These models are designed to assist in Financial Planning and Analysis (FP&A) and equity research practices.

## Repository Structure

The project is divided into several key directories based on the financial modeling methodology:

### 1. [DCF (Discounted Cash Flow)](./DCF/README.md)
Contains models and tutorials for performing Discounted Cash Flow valuations, used to estimate the value of an investment based on its expected future cash flows.
- **`2022.07.24 - DCF Tutorial Empty Excel.xlsx`**: Blank template for DCF practice and learning.
- **`DCF_Tutorial_Practice_File.xlsx`**: Practice file accompanying DCF tutorials.
- **`Start File DCF Exercise_v2.xlsx`**: Structured exercise for building a DCF model from scratch.

### 2. [DDM (Dividend Discount Model)](./DDM/README.md)
Focuses on valuing a company's stock price based on the theory that its stock is worth the sum of all of its future dividend payments, discounted back to their present value.
- **`Dividend discount model.xlsx`**: Standard DDM template/model.
- **`ITC DIVIDEND DISCOUNT MODEL.xlsx`**: Specific DDM valuation applied to ITC Limited.

### 3. [Relative Valuation](./Relative%20Valuation/README.md)
Contains models comparing companies' valuations based on multiples (e.g., P/E, EV/EBITDA) against peers in the same sector.
- **`Relative Valuation.xlsx`**: General relative valuation model across various comparable companies.
- **`Relative valuation Pharma industry.xlsx`**: Industry-specific relative valuation tailored for the Pharmaceutical sector.
- **`Annual reports/`**: A subdirectory containing PDF annual reports for various major Indian companies (Apollo Hospitals, Aster DM, Britannia, Fortis, Godrej Consumers, HUL, ITC, Max Hospitals, Narayana Health, Nestle, Tata Consumer). These reports serve as the primary source of historical financial data for the models.

### 4. [Risk Calculation](./Risk%20Calculation/README.md)
Models dedicated to evaluating stock performance, fundamentals, and risk against market benchmarks like the Nifty 50.
- **`Multi Industry Stock Analysis.xlsx`**: Comparative analysis and risk metrics across multiple industries.
- **`Stocks Vs. Nifty 50 Fundamentals.xlsx`**: Fundamental comparison of specific stocks against the Nifty 50 index.
- **`Stocks Vs. Nifty 50 Ratio Analysis.xlsx`**: Detailed financial ratio analysis comparing individual stocks to the broader market index.
- **`Stocks Vs. Nifty 50.xlsx`**: High-level comparison and risk assessment of selected stocks against Nifty 50.

### 5. [WACC (Weighted Average Cost of Capital)](./WACC/README.md)
Calculations for determining a firm's cost of capital, an essential input for discounting cash flows in DCF and other valuation models.
- **`WACC Wipro.xlsx`**: Detailed WACC calculation and component breakdown for Wipro Limited.

## Getting Started

To utilize these models:
1. Clone or download this repository to your local machine.
2. Open the desired `.xlsx` file using Microsoft Excel (recommended) or a compatible spreadsheet application.
3. Review the assumptions, hardcoded inputs, and formulas. You can reference the `Annual reports` folder under `Relative Valuation` to verify historical financial figures.

## Disclaimer

These models are intended for educational, practice, and reference purposes for FP&A and Equity Research. Always verify the formulas and update inputs with current market data before using them for actual investment decisions.
